# 🔍 Sensitive File Scanner — Firefox Extension

Расширение для Firefox, которое **мгновенно проверяет веб-сайт** на наличие общеизвестных уязвимых файлов и директорий.  
Предназначено для пентестеров, разработчиков и всех, кто заботится о безопасности.

## 🚀 Возможности

- **Два режима сканирования**:
  - `Проверить сайт` — сканирует корень домена (например, `https://example.com/.env`)
  - `Проверить текущий путь` — сканирует относительно текущей директории (например, если вы на `https://example.com/admin/`, проверит `https://example.com/admin/.git/config`)
- **Более 230 встроенных путей** (и легко расширяется)
- **Прогресс-бар** с отображением количества проверенных путей
- **Цветовая индикация** статусов ответа (зелёный — 200, красный — 500 и т.д.)
- **Фильтрация** неинформативных ответов (404 не показываются)
- **Загрузка списка путей из внешнего файла `wordlist.txt`** — можно редактировать без переустановки расширения
- Работает полностью локально, без внешних сервисов
- Поддержка **Firefox** (Manifest V3)
